### Enable Domain-Driven Design with Gamification

Domain-Driven Design Cards offer valuable methodological support for a range of strategic and 
tactical Domain-Driven Design workshops, and they're available for free. This initiative is a collaborative 
project with [nilsbert](https://github.com/nilsbert). For more information about us, you can explore the [website](www.architecture-enablers.de).

With Domain-Driven Design Cards, you can engage in activities such as the Context Mapping Game, 
the Strategic Classification Game, the Bounded Context Game, and the Architecture Game. If you're looking for a comprehensive guide 
on how to use them, please consult the following sections.

### Enable Domain-Driven Design mit Gamification

Domain-Driven Design Cards bieten wertvolle methodische Unterstützung für eine Reihe von strategischen und 
taktischen Domain-Driven Design Workshops und sind kostenlos verfügbar. Diese Initiative ist ein gemeinschaftliches 
Projekt mit [nilsbert](https://github.com/nilsbert). Weitere Informationen über uns findest du auf der [Website](www.architecture-enablers.de).

Mit Domain-Driven Design Cards kannst du Aktivitäten wie das Context Mapping Game, 
das Strategic Classification Game, das Bounded Context Game und das Architecture Game durchführen. Wenn du einen umfassenden Leitfaden 
zur Verwendung suchst, konsultiere bitte die folgenden Abschnitte.
